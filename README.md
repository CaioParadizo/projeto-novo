# Portfólio - Caio Paradizo

Portfólio profissional desenvolvido para apresentar minha trajetória, projetos, experiências, habilidades técnicas e formas de contato.

O projeto foi construído com foco em desempenho, responsividade, acessibilidade e experiência do usuário, reunindo informações profissionais, integração com GitHub e currículo para recrutadores e empresas.

## Funcionalidades

* Alternância de idioma (Português / Inglês)
* Alternância entre tema Dark e Light
* Layout responsivo para desktop e dispositivos móveis
* Integração com GitHub para exibição de informações do perfil
* Download do currículo em PDF
* Seção de cursos, formação acadêmica e experiências profissionais
* Links rápidos para contato e redes profissionais

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Git
* GitHub Pages
