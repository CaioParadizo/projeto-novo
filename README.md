# Projeto Dia dos Namorados

Mini página estática com:

- Carrossel de 3 fotos com frases de efeito
- Contador interativo de dias, horas, minutos e segundos
- Seletor de áudio para tocar sua música favorita
- Atualização do contador a cada segundo

## Como usar

1. Abra `index.html` no navegador.
2. Ajuste a data de início do relacionamento se necessário.
3. Clique em "Selecionar música" e escolha um arquivo de áudio.
4. Use os controles para reproduzir ou pausar a música.
